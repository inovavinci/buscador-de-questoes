# Domain Framework: Vestibular Question Curation

## 1. Retrieval Strategy
- **Source Identification:** Prioritize official portals (INEP), reputable question banks (Ex: Projeto Medicina, Toda Matéria), and school repositories.
- **Enunciado Extraction:** Copy full text, preserving formatting and special characters.
- **Image Handling:** Identify image placeholders or URLs and preserve their context for the final document.
- **Answer Key:** Cross-reference question ID or text with answer keys to ensure a match.

## 2. Classification Logic
- **Easy:** Direct application of concepts, basic recall, simple calculations.
- **Medium:** Requires conceptual integration, analysis of multiple variables, or moderate calculations.
- **Hard:** Advanced problem-solving, multidisciplinary reasoning, complex calculations, or deep critical analysis.

## 3. Formatting Standards
- **Header:** Board Name (Banca), Year, Location (if applicable).
- **Body:** Numbered question, clear differentiation between choices (A, B, C, D, E).
- **Footer:** Answer Key and Difficulty tag.
